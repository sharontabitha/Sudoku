# Sudoku
Tugas Algoritma Pemrograman 
    
    Program ini merupakan permainan Sudoku yang dibuat menggunakan Python dengan menerapkan algoritma Backtracking untuk menyelesaikan puzzle. Aplikasi ini menghasilkan papan Sudoku yang mudah dan memungkinkan pemain mengisi angka yang kosong sambil memeriksa kebenaran setiap input secara langsung. Program juga dilengkapi dengan beberapa fitur seperti batas kesalahan, timer permainan, pencatatan waktu terbaik (best time), serta animasi penyelesaian Sudoku yang memperlihatkan bagaimana algoritma backtracking bekerja langkah demi langkah dalam menemukan solusi yang benar.
      Program ini dibuat sebagai contoh penerapan algoritma rekursif dan teknik backtracking dalam menyelesaikan permasalahan berbasis constraint seperti Sudoku.

Untuk run coding sebaikan import tkinter terlebih dahulu.
